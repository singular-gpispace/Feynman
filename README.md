# feynman

The package Feynman computes complete set of IBP identities of Feynman integral associated to a given Feynman graph using module intersection method. Paralleization of the computation is given as an otion which will be used gpi-space for such computations. Grynman is based on the computer algebra system OSCAR and is provided as a package for the Julia programming language.
